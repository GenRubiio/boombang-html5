## SonarQube:

### Locally with the boomania repository with docker:
#### https://git.basetis.com/devops/sonarqube

```
Start docker from devops repository
docker compose up -d

Execute scan
docker run --rm --network=host -e SONAR_HOST_URL="http://localhost:9000" -e SONAR_LOGIN="<your_access_token>" -v "${PWD}:/usr/src" sonarsource/sonar-scanner-cli -Dsonar.projectKey="<your_project_key>"
```

#### In Mac M1, with sonar-scanner installed by brew:
```
/opt/homebrew/Cellar/sonar-scanner/5.0.1.3006/bin/sonar-scanner -Dsonar.projectKey="<your_access_token>" -Dsonar.sources=. -Dsonar.host.url=http://localhost:9000 -Dsonar.login="<your_project_key>"
```

### Configure sonar-project.properties
```
Change projectKey, projectName and login variables.
Now execute: sonar-scanner
```

## Sail:

### How to start project with docker (sail)

* First install:
```
composer require laravel/sail --dev
php artisan sail:install
```

* Execute command into the project folder:
````
./vendor/bin/sail up
````

* Create an alias for sail if needed:
````
alias sail='[ -f sail ] && bash sail || bash vendor/bin/sail'
sail up
````

* If you prefer start Docker containers in background:
````
sail up -d
````
